# vue-multiple-single-pages

## 说明
Starter use Vue-cli4

该项目将Vue的多页和单页整合到一起，如果这符合你的开发需求，可以直接再此基础上进行拓展

#### 适用场景
产品由多个相对独立，却又有部分相关的子产品组合而成的集成仓库

可满足：
1.多个入口配置，保证各个子产品有自己独立的配置
2.子产品内部可以使用vue基本的单页应用配置
